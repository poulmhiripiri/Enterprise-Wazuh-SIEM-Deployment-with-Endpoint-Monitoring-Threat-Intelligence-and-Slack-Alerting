# 05 - VirusTotal Threat Intelligence Integration

## Objective

Use VirusTotal to enrich Wazuh detections with external threat intelligence.

## Use Case

When suspicious files or hashes are detected, Wazuh can query VirusTotal to determine whether the file or indicator has a known malicious reputation.

## Security Value

VirusTotal enrichment supports:

- Faster analyst triage
- File reputation validation
- Malware investigation
- Improved detection confidence
- Better prioritisation of suspicious activity

## Public Repository Safety Note

Do not publish VirusTotal API keys. Redact all API keys and token values before uploading screenshots or configuration files to GitHub.
