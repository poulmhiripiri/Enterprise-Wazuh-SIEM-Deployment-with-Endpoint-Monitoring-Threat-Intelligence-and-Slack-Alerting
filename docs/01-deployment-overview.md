# 01 - Wazuh Deployment Overview

## Objective

Deploy Wazuh as a centralised SIEM platform for an SME-style environment, providing security visibility across Windows and Linux endpoints.

## Scope

- Wazuh Manager
- Wazuh Dashboard
- Wazuh agents
- Windows endpoint monitoring
- Linux endpoint monitoring
- Security event validation
- Dashboard review and alert triage

## Skills Demonstrated

- SIEM platform deployment
- Virtualised lab implementation
- Endpoint telemetry collection
- Security event validation
- Operational troubleshooting
