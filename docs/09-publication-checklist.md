# 09 - Public GitHub Publication Checklist

Before uploading this repository publicly, confirm the following:

- [ ] No public IP addresses, private IPs, hostnames, or usernames are exposed unless intentionally shared.
- [ ] VirusTotal API keys are removed or redacted.
- [ ] Slack webhook URLs, bot tokens, and app credentials are removed or redacted.
- [ ] Default passwords are not shown in screenshots.
- [ ] Email addresses and workspace invitation links are removed.
- [ ] Screenshots are renamed clearly and referenced in the README.
- [ ] The repository description is concise and recruiter-friendly.
- [ ] The README explains business value, not only technical steps.
- [ ] Any raw lab files are placed in an evidence folder and reviewed before upload.

## Recommended Screenshot Redaction Areas

- Browser address bars
- API key fields
- Slack app credentials
- Internal IP addresses
- Local usernames
- Email addresses
- Default admin credentials
